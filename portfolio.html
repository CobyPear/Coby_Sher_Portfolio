Dish Tracker can be found here

Dish Tracker is a web application designed to help restaurant goers keep track of dishes they've had and dishes they'd like to try on their next visit to a restaurant. It uses the user's location data to pull up nearby restaurants and their menus via the US Restaurant Menu API
